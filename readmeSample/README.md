
# Tic tact Toe
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
        
## Description
Tic tact Toe is a two player game in which player mark X's and O's on a 3x3 grid to get three in a row..
